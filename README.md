# LVGL移植
## 环境要求
+ ESP-IDF 4.4.7  
+ ESP32S3 N16R8
+ ST7796 SPI显示器,分辨率为320*480
+ GT911电容触摸芯片
+ LVGL 8.3
+ lvgl_esp32_drivers  
---  
本仓库是将LVGL到ESP32S3上，使用的开发环境是ESP-IDF 4.4.7，理论上4.4的版本应该都可以使用  
使用维可思的3.5寸TFT电容开发板
